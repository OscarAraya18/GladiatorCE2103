◆ colocarTorres()

void Matriz::colocarTorres ( )

Por cada iteracion se deben colocar 3 torres en la matriz. Primero se genera una posicion aleatoria donde colocar esta torre. Luego se valida que esta posicion no sea el punto de inicio, el punto final o que ya cuente con una torre. Finalmente se evalua que esta posicion no bloquee la unica ruta disponible para que los gladiadores salgan de la zona de intimidacion
