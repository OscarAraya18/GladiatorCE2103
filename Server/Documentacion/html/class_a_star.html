◆ destinoEncontrado

bool AStar::destinoEncontrado

Es un flag que indica si el algoritmo A* ha encontrado al menos una ruta posible desde la posicion inicial hasta la final. Se utiliza para validar si cuando se colocan torres queda al menos una ruta para que los gladiadores puedan cruzar
